TITLE: Pipeline Health Monitor
DESCRIPTION: Observes MES manufacturing run lifecycle signals and emits structured health scores per run, surfacing stalled agents, entropy spikes, and worker absence patterns in real time.
PLUGIN: Ichor.Plugins.PipelineHealthMonitor
SIGNAL_INTERFACE: Subscribes to run_started, run_complete, agent_heartbeat, agent_stalled, worker_absent signals; emits health_score_updated and pipeline_degraded signals.
TOPIC: plugin:pipeline_health_monitor
VERSION: 0.1.0
FEATURES: per-run health scoring, stalled agent detection, entropy spike alerting, worker absence tracking, real-time PubSub broadcast of health deltas
USE_CASES: dashboard health panel, Archon entropy healing trigger, ops alerting, run post-mortem analytics
ARCHITECTURE: GenServer per active run (started via DynamicSupervisor on run_started signal); accumulates signal events into a health state map; periodically emits health_score_updated via Phoenix.PubSub; terminates on run_complete and emits final score. Supervised under Ichor.Infrastructure.
DEPENDENCIES: Ichor.SignalBus, Ichor.Factory.Projects.Runner, Ichor.Infrastructure, Phoenix.PubSub
SIGNALS_EMITTED: health_score_updated, pipeline_degraded, run_health_finalized
SIGNALS_SUBSCRIBED: run_started, run_complete, agent_heartbeat, agent_stalled, worker_absent, entropy_spike
