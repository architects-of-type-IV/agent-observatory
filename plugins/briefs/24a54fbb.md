# Plugin Brief: 24a54fbb

TITLE: SignalCorrelator
DESCRIPTION: Cross-agent signal correlation engine that detects timing patterns, bottlenecks, and coordination failures across multi-agent pipelines. Subscribes to all agent signal topics and emits correlation events when statistical anomalies or causal chains are detected.
PLUGIN: Ichor.Plugins.SignalCorrelator
SIGNAL_INTERFACE: Subscribes to signals/bus events; emits correlated findings via plugin:signal_correlator topic
TOPIC: plugin:signal_correlator
VERSION: 0.1.0
FEATURES: cross-agent signal windowing, causal chain detection, bottleneck scoring, correlation event emission, configurable window size, per-agent latency tracking
USE_CASES: diagnosing stalled multi-agent pipelines, identifying entropy accumulation root causes, detecting coordination deadlocks, observability for DAG pipeline runs
ARCHITECTURE: GenServer holding a sliding time-window buffer of recent signal events per agent; on each new signal, runs correlation pass over the buffer; emits :correlation_detected signal when score exceeds threshold; supervised under Ichor.Infrastructure.LifecycleSupervisor
DEPENDENCIES: Ichor.Signals.Bus, Ichor.Signals.EntropyTracker, Ichor.Projector.SignalBuffer, Ichor.MemoryStore
SIGNALS_EMITTED: :correlation_detected, :bottleneck_identified, :pipeline_stall_detected
SIGNALS_SUBSCRIBED: :agent_signal, :entropy_spike, :agent_watchdog_alert, :session_lifecycle
