# Section 5.4: Scheduler and Forensic Inspector Views

## Purpose

This section implements `SchedulerComponents` and `ForensicComponents` covering FR-12.9 and FR-12.10. The Scheduler view renders the Cron Job Dashboard, DLQ panel, and Heartbeat Monitor. The Forensic Inspector view renders a full-text queryable Message Archive, Cost Attribution panel, Security webhook log panel, Policy Engine rule manager, and collapsible sub-panels for Errors, Analytics, and Timeline.
