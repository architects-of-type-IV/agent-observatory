# Section 4.2: CronScheduler & DB Schema

## Purpose

This section implements the `Observatory.Gateway.CronScheduler` GenServer with its `schedule_once/3` public API, creates the `cron_jobs` SQLite table migration, implements startup recovery for unfinished jobs, and wires the SchemaInterceptor extension that translates `schedule_reminder` tool calls into `CronScheduler.schedule_once/3` invocations.
