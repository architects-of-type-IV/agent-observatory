# Task 3.3.2: Node State Color Mapping

## Goal

The Canvas renderer must translate each node's `state` field to an exact hex fill color on every animation frame. The `alert_entropy` state additionally requires a periodic flashing animation. Any unrecognised or absent state must fall back to the idle color `#6b7280` without producing a runtime error.
