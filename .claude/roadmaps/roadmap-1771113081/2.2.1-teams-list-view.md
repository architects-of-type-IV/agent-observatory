# Story 2.2.1: Teams List View

## Delivers
A :teams view component showing all running teams as rows with status, health, member count, task progress, and "Inspect" button.

## Acceptance Criteria
- Component renders team rows: name, member count, health dot, task progress, status
- "Inspect" button sends phx-click="inspect_team" phx-value-name={name}
- Dead teams shown greyed out
- Empty state when no teams
- Module under 300 lines
- `mix compile --warnings-as-errors` passes
