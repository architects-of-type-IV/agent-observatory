# Task 1.3.2: JSON Deserialization

## Goal

The `from_json/1` function provides a named entry point for the Gateway controller to cast a decoded JSON map (string-keyed, as produced by `Jason.decode!/1` or `Plug.Parsers`) into a validated `%DecisionLog{}` struct. It wraps `changeset/2` and `apply_changes/1`, returning a two-tuple `{:ok, struct}` or `{:error, changeset}` so callers can pattern-match on the result without inspecting changeset internals directly.
