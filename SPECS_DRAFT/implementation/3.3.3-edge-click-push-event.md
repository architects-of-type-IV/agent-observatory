# Task 3.3.3: Edge Click pushEvent Contract

## Goal

`handleClick(e)` must translate a canvas mouse event into either an `edge_selected` pushEvent, a `node_selected` pushEvent, or no action. Node hit detection takes precedence over edge detection when coordinates could match both.
