Phoenix LiveView — Signals Page Implementation Spec

Module Structure

  • IchorWeb.SignalsLive — root LiveView, owns feed state and filter assigns
  • IchorWeb.Components.SignalCatalog — stateless functional component; renders accordion catalog
  • IchorWeb.Components.SignalEmitter — stateful live component; handles emit form state
  • IchorWeb.Components.SignalFeedRow — functional component; renders one feed row with expansion
  • IchorWeb.Components.CategoryPill — functional; renders a filter pill

Key Assigns (SignalsLive)

@signals           [%{id, name, category, data, ts, level}]  # ring buffer, max 500
@filtered_signals  derived from @signals + @filters
@filters           %{search: "", categories: [], signal_name: nil}
@paused            boolean
@category_rates    %{atom => float}    # signals/min per category
@active_preset     atom
@expanded_row_id   string | nil

handle_info/2 — Live Feed Ingest

def handle_info({:signal, name, data}, socket) do
  if socket.assigns.paused do
    {:noreply, socket}
  else
    entry = %{
      id:       System.unique_integer([:positive, :monotonic]),
      name:     name,
      category: Ichor.Signals.Catalog.lookup!(name).category,
      data:     data,
      ts:       DateTime.utc_now(),
      level:    classify_level(name)
    }
    signals = [entry | socket.assigns.signals] |> Enum.take(500)
    {:noreply, assign(socket, signals: signals, filtered_signals: apply_filters(signals, socket.assigns.filters))}
  end
end

handle_event/3 Handlers

  • "toggle_category" — toggle category in @filters.categories
  • "search_feed" — update @filters.search, recompute @filtered_signals
  • "filter_signal" — set @filters.signal_name from catalog click
  • "toggle_pause" — flip @paused
  • "clear_feed" — set @signals = []
  • "apply_preset" — set @filters to preset values
  • "expand_row" — set @expanded_row_id
  • "emit_signal" — call Ichor.Signals.emit/2 with validated payload
  • "catalog_search" — filter catalog accordion (client-side via JS hook)

}

Rate Calculation (category_rates)

Tick every 10s via Process.send_after(self(), :recalc_rates, 10_000). Count signals in last 60s per category, divide by 1.

JS Hook: CatalogSearch

Client-side filtering of .category-group and .signal-item elements. Uses phx-hook="CatalogSearch" on the search input. Avoids unnecessary server round-trips for catalog accordion filtering.
