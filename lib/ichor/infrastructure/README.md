# Ichor.Infrastructure

This folder is for implementation infrastructure, not product/domain modeling.

Put code here when it answers one of these questions:

- How do we talk to tmux?
- How do we talk to MCP?
- How do we launch or supervise host-side processes?
- How do we sync files across app and repo boundaries?
- How do we integrate with external systems without creating a new domain?

Do not put Workshop, Factory, Signals, or Archon business concepts here.
