TITLE: Vigilance: Entropy-Based Fleet Anomaly Detector
DESCRIPTION: Computes Shannon entropy of per-session signal-type frequency distributions to establish rolling behavioral baselines, then applies Z-score deviation scoring to detect anomalous agent behavior before it escalates to failure. Combines entropy management (baseline tracking) and anomaly detection (deviation scoring) into a single lightweight GenServer subsystem.
SUBSYSTEM: Ichor.Vigilance
SIGNAL_INTERFACE: Subscribes to :fleet and :activity signal categories via Ichor.Signals.subscribe/1; emits :anomaly_detected, :baseline_established, :entropy_spike, :entropy_normalized under the :vigilance category; topic subsystem:vigilance
TOPIC: subsystem:vigilance
VERSION: 0.1.0
FEATURES: per-session Shannon entropy computation over signal-type histograms, EMA-based rolling baseline (configurable alpha), Z-score anomaly scoring with configurable sigma threshold, ETS ring buffer of anomaly history per session, automatic baseline reset on session restart signal, entropy health score per agent for dashboard display, configurable window size and sensitivity, heartbeat-driven sweep for stale sessions
USE_CASES: detect runaway tool-call loops before they exhaust budget, flag sessions with sudden signal-type shifts indicating stuck or confused agents, surface entropy regression after agent recovery to confirm healthy return to baseline, alert on silent agents whose entropy drops to zero indicating a frozen session, feed anomaly scores into Ichor.Nervous.Correlator as first-class signals for cascade detection
ARCHITECTURE: Single supervised GenServer (Ichor.Vigilance) owns three ETS tables: SessionHistogram (session_id -> signal-type frequency map for current window), BaselineTable (session_id -> %Baseline{mean_entropy, ema_variance, window_count}), AnomalyLog (session_id -> ring buffer of last 50 anomaly events). On each incoming signal: (1) update SessionHistogram for the session, (2) on window boundary (configurable, default 30s) compute Shannon entropy H = -sum(p * log2(p)) over the histogram, (3) update BaselineTable via EMA: new_mean = alpha*H + (1-alpha)*old_mean, (4) compute Z-score = abs(H - mean) / sqrt(variance), (5) if Z > threshold emit :anomaly_detected with session_id, entropy, z_score, and signal_snapshot. Heartbeat sweep evicts sessions with no signals in >10 minutes. Baseline is reset when a fleet.agent.started signal arrives for a known session_id.
DEPENDENCIES: Ichor.Signals, Ichor.Signals.Catalog, Ichor.Signals.Bus, Ichor.Fleet.AgentProcess, Ichor.Fleet.FleetSupervisor, :ets, :math
SIGNALS_EMITTED: :vigilance_anomaly_detected, :vigilance_baseline_established, :vigilance_entropy_spike, :vigilance_entropy_normalized, :vigilance_session_frozen
SIGNALS_SUBSCRIBED: :fleet, :activity
