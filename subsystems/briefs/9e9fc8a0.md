TITLE: Adaptive Entropy Balancer
DESCRIPTION: A fleet health subsystem that continuously measures Shannon entropy across agent signal flows and applies adaptive weighted load balancing to prevent coordination breakdown and cascade failures in the Ichor IV agent control plane.
SUBSYSTEM: Ichor.Subsystems.AdaptiveEntropyBalancer
SIGNAL_INTERFACE: subscribes to all Signals.Catalog categories via Signals.Bus; responds to heartbeat ticks for periodic recompute; emits health and rebalance signals
TOPIC: subsystem:adaptive_entropy_balancer

FEATURES: Shannon entropy measurement per agent and category, 60s sliding window signal analysis, weighted round-robin task routing, backpressure detection via queue-fill ratio, EMA-smoothed per-agent health scores, cascade risk alerting, fleet-wide entropy dashboard feed
USE_CASES: Detect agent overload before cascade propagates, rebalance task distribution under skewed fleet load, identify communication disorder in multi-agent coordination, surface entropy hotspots on the Ichor dashboard, auto-escalate high-entropy agents to HITL review
