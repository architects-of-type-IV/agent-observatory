# Refactor Audit (2026-03-19)

Audit of the ichor host app. The original inventory (~80% stale) has been stripped;
only active findings and resolved items are preserved here.

## Active Findings

### Boundary Violations

#### HIGH: Nested `defmodule Node` inside `Ichor.Mesh.CausalDAG`

- File: `lib/ichor/mesh/causal_dag.ex:14`
- `defmodule Node` is nested inside `CausalDAG`. It must be extracted to its own file
  at `lib/ichor/mesh/causal_dag/node.ex`.

#### HIGH: `Ichor.Workshop.TeamSync` calls raw Ash functions

- File: `lib/ichor/workshop/team_sync.ex`
- Lines 13, 15, 23, 51 call `Ash.read/1`, `Ash.destroy/1`, `Ash.Changeset.for_create/3`,
  `Ash.create/1` directly on `TeamMember`. These must go through the `Ichor.Workshop`
  domain code_interface.

### Dead Code

#### LOW: `Ichor.Signals.ProtocolTracker` — dead `command_queue` field

- File: `lib/ichor/signals/protocol_tracker.ex:167`
- `command_queue: %{total_pending: 0}` is initialized in state but never populated. The field
  always emits zero values. Remove the field and its emission in `compute_stats/0`.

## Resolved (DONE)

The following findings from the original audit have been addressed through the
2026-03-19 through 2026-03-22 architecture refactor.

  (`fleet_supervisor.ex`, `host_registry.ex`, `agent_launch.ex`, `registration.ex`, `cleanup.ex`).
- **DONE** — `Mes.*` namespace dissolved. Manufacturing execution moved to `Factory.*`
  domain (`lib/ichor/factory/`).
- **DONE** — `Genesis.*` namespace dissolved. Genesis pipeline moved to `Factory.*`.
- **DONE** — `Dag.*` namespace dissolved. DAG runtime merged into `Factory.*`
  (`pipeline_compiler.ex`, `pipeline_task/`, Oban workers).
- **DONE** — `Gateway.*` namespace dissolved. Routing and delivery moved to
  `Infrastructure.*` and `Signals.*`.
- **DONE** — `SwarmMonitor.*` dissolved. Monitoring responsibilities distributed to
  signal subscribers and Oban workers.
- **DONE** — `AgentTools.*` domain dissolved. MCP tools reorganized.
- **DONE** — `DecisionLog` nested struct modules (`Meta`, `Identity`, `Cognition`, `Action`,
  `StateDelta`, `Control`) extracted. `decision_log/helpers.ex` submodule present at

  superseded by ichor_contracts or removed).
- **DONE** — `HitlEvents` nested modules resolved.
- **DONE** — `Mes.TeamSpawner` 23-line facade removed.
- **DONE** — `Fleet.Lifecycle` 32-line delegate removed.
- **DONE** — `Fleet.SessionEviction` / `Fleet.Analysis.SessionEviction` duplication resolved;
  canonical at `lib/ichor/workshop/analysis/session_eviction.ex`.
- **DONE** — `EventBuffer` and `HeartbeatManager` absorbed into `Events.Runtime`.
- **DONE** — `AgentMonitor` and `NudgeEscalator` dissolved; superseded by signal subscribers.
